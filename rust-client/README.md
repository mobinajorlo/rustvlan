# RustNet Virtual LAN Client & Coordinator Guide (Farsi & English)

این پوشه شامل یک کلاینت کاملاً بومی (Native CLI Client) به زبان **رست (Rust)** است که با هماهنگ‌کننده شبکه و وب‌سوکت در ارتباط است. شما می‌توانید این کلاینت را در ویندوز خود کمپایل کرده و به یک فایل `.exe` تک و سبک تبدیل کنید!

---

## 🟢 بخش اول: راهنمای راه‌اندازی سرور هماهنگ‌کننده (Linux / VPS)
پلتفرم هماهنگ‌ساز با زبان Node.js نوشته شده است. مراحل زیر را بر روی سرور لینوکس ابونتو (Ubuntu) خود اجرا کنید:

### ۱. نصب پیش‌نیازها بر روی سرور لینوکس:
ابتدا پکیج کنترلر Node.js را نصب کنید:
```bash
sudo apt update
sudo apt install -y nodejs npm
```

### ۲. ساخت پوشه پروژه و فایل سرور:
فایل سرور از پیش آماده شده به نام `standalone-server.js` در فایل‌های دانلودی شماست. محتویات آن را با دستور زیر بر روی لینوکس خود بسازید یا کپی کنید:
```bash
mkdir rustnet-server
cd rustnet-server
# فایل standalone-server.js را درون این پوشه آپلود یا کپی کنید.
```

### ۳. نصب پکیج‌های وب سرور:
پکیج وب‌سوکت را نصب کنید:
```bash
npm install ws
```

### ۴. اجرای سرور (Background Execution):
برای اینکه پس از بستن ترمینال لینوکس، سرور همچنان آنلاین بماند، از پکیج `pm2` استفاده کنید:
```bash
sudo npm install -g pm2
PORT=3000 pm2 start standalone-server.js --name "rustnet-server"
```
*اکنون سرور شما بر روی آدرس `ws://vps_ip_address:3000` به صورت ۲۴ ساعته اجرا شده و آماده هماهنگ‌سازی اتصالات Rust است!*

## 🔵 بخش دوم: نحوه کامپایل کلاینت رست و خروجی `exe` در ویندوز (با Visual Studio)

چون شما **Visual Studio** را نصب دارید، در حال حاضر ابزار عالی برای کامپایل کدهای بومی سیستمی (C++ & Rust Toolchain) در اختیار دارید. مراحل زیر را در ویندوز خود برای ایجاد خروجی `.exe` انجام دهید:

### ۱. نصب ابزار Rust در ویندوز:
۱. به آدرس رسمی [rustup.rs](https://rustup.rs/) بروید و فایل نصاب ویندوز (`rustup-init.exe`) را دانلود و اجرا کنید.
۲. در محیط ترمینال باز شده، گزینه `1` (پیش‌فرض) را برای نصب Rust با زنجیره ابزار Visual Studio MSVC انتخاب کرده و نصب را نهایی کنید.
۳. پس از پایان نصب، یک بار سیستم یا برنامه ترمینال خود را ببندید و باز کنید.

### ۲. کامپایل کلاینت رست:
۱. پوشه پروژه را با باز کردن منوی بالای AI Studio و انتخاب دکمه **Export to ZIP** دانلود کنید و به سیستم ویندوزی خود منتقل و استخراج (Unzip) کنید.
۲. یک ترمینال (یا CMD / PowerShell) درون پوشه `rust-client` (جایی که فایل `Cargo.toml` قرار دارد) باز کید.
۳. دستور زیر را برای بیلد نهایی و بهینه اجرا کنید:
```bash
cargo build --release
```

### ۳. دریافت فایل نهایی `.exe`:
بخش کامپایل پس از چند ثانیه به پایان می‌رسد. به مسیر زیر در ویندوز بروید:
```text
rust-client/target/release/
```
درون این پوشه، فایل نهایی تولید شده سبک به نام **`rustnet-client.exe`** را مشاهده خواهید کرد!

### ۴. نحوه استفاده از کلاینت:
کافیست فایل `.exe` بدست آمده را اجرا کرده، اطلاعات زیر را وارد کنید و بلافاصله به شبکه متصل شوید:
- **WebSocket URL**: آدرس سرور لینوکس خود را وارد کنید (مثال: `ws://YOUR_VPS_IP:3000`)
- **Nickname**: آی‌دی یا اسم دلخواه برای لابی بازی وارد کنید (مثال: `Arash_Gamer`)
- **Lobby Room ID**: نام رومی که مایلید همتایانتان به آن متصل شوند (مثال: `survival-room`)

*پس از اتصال، با دستور `/host Minecraft 19132` می‌توانید یک بازی مجازی را روی شبکه شبیه‌سازی کنید تا بقیه بازیکنان متصل در روم آن را شناسایی کنند.*

## 🖥️ بخش سوم: نحوه کامپایل نسخه گرافیکی و ظاهری (GUI Desktop Client) به همراه تمامی دکمه‌ها و منوها

اگر مایلید به جای محیط سیاه متنی (CLI)، **دقیقاً رابط کاربری پیشرفته، دکمه‌ها، پنل چت گرافیکی، لایو سابنت آی‌پی و منوها** را به صورت یک برنامه محلی مستقل و بسیار سبک در ویندوز داشته باشید، ما بستر **Tauri 2** را برای شما درون پروژه پیکربندی کرده‌ایم. تائوری با استفاده از وب‌ویو نیتیو ویندوز و بستر قدرتمند رست، برنامه فرانت‌اند را به یک فایل سبک `.exe` تبدیل می‌کند.

ترکیب Visual Studio، ابزار Rust و پکیج Node.js در سیستم شما، برای به دست آوردن خروجی عالی و سبک هم‌اکنون آماده است:

### مراحل گام‌به‌گام ساخت فایل `exe` گرافیکی:

#### ۱. رفتن به مسیر اصلی پروژه:
ابتدا یک ترمینال (یک پنجره CMD یا PowerShell) در **پوشه اصلی کل پروژه** (جایی که فایل‌های `package.json` و `src/` قرار دارند) باز کنید.

#### ۲. نصب وابستگی‌های برنامه:
دستور زیر را در ترمینال بنویسید تا بخش فرانت‌اند و کلاینت تائوری دانلود و نصب شوند:
```bash
npm install
```

#### ۳. تولید آیکون‌های ویندوزی (مرحله اختیاری):
برای ایجاد خودکار آیکون‌های استاندارد ویندوز، لینوکس و مک برای فایل اجرایی خود، این دستور ساده تائوری را اجرا کنید تا پکیج آیکون کامل را برایتان بسازد:
```bash
npx tauri icon ./src/assets/react.svg
```
*(اگر می‌خواهید آیکون دلخواه خود را قرار دهید، تصویر خود را با فرمت PNG جایگزین فایل react.svg کنید و دستور بالا را بزنید).*

#### ۴. کامپایل نهایی و گرفتن فایل `.exe` گرافیکی:
حال با زدن دستور زیر، تائوری ابتدا کدهای گرافیکی React شما را کامپایل کرده و سپس با استفاده از زنجیره ابزار Rust (کامپایلر نیتیو ویندوز ام‌اس‌وی‌سی که داخل ویژوال استودیو است) آن را به یک فایل اسمبلی دات‌اگزه تک و کاملاً مستقل تبدیل می‌کند:
```bash
npm run tauri build
```

#### ۵. دسترسی به برنامه گرافیکی شما:
بخش کامپایل پس از چند دقیقه کامل می‌شود. فایل جادویی و زیبای خود را در مسیر زیر دریافت خواهید کرد:
```text
src-tauri/target/release/RustNet.exe
```
یا درون فولدر باندل نصب‌کننده:
```text
src-tauri/target/release/bundle/msi/RustNet_0.1.0_x64_en-US.msi
```

با اجرای **`RustNet.exe`**، شما یک نرم‌افزار فوق‌العاده شیک، بسیار سبک (کمتر از ۵ مگابایت!) و واکنش‌گرا با تمامی دکمه‌ها، تنظیمات فرکانس، انتقال زنده فایل‌ها، مدیریت روم و شبیه‌ساز کارت شبکه مجازی خواهید داشت که با سرعت عالی در ویندوز اجرا می‌شود!

## 🛠️ رفع خطای احتمالی کامپایل ویندوز (`RC2176: old DIB` یا `resource.rc`)

در برخی نسخه‌های ویندوز و ویژوال استودیو، ممکن است در خطوط آخر فرآیند بیلد با خطایی مشابه زیر روبرو شوید:
`error RC2176: old DIB in C:\...\icons\icon.ico; pass it through SDKPAINT`

### علت خطا:
برنامه `RC.EXE` (کامپایلر منابع مایکروسافت در ویژوال استودیو) بسیار قدیمی است و نمی‌تواند تصاویر با فرمت فشرده‌ی جدید (PNG-compressed) داخل فایل‌های `.ico` مدرن را که توسط ابزارهای خودکار تولید می‌شوند بخواند.

### راهکار حل سریع (فقط در ۲ مرحله):

#### روش اول: جایگزینی فایل آیکون با یک نسخه کاملاً استاندارد و غیرفشرده
۱. یکی از آیکون‌های فرمت PNG موجود در پوشه را (مثلا `src-tauri/icons/128x128.png`) کپی کنید.
۲. وارد یکی از سایت‌های رایگان تبدیل فرمت آنلاین مانند **[CloudConvert](https://cloudconvert.com/)** یا **[ICOConvert](https://icoconvert.com/)** یا **[Convertio](https://convertio.co/)** شوید.
۳. تصویر PNG کپی شده را به فرمت `.ico` جدید تبدیل کنید (ترجیحاً در ابعاد استاندارد مانند `48x48` یا `128x128` و فرمت معمولی/غیر فشرده).
۴. فایل دانلود شده از سایت را با نام دقیق **`icon.ico`** نام‌گذاری کرده و جایگزین فایل خراب در مسیر زیر کنید:
   `src-tauri/icons/icon.ico`
۵. مجدداً دستور زیر را اجرا کنید تا برنامه بدون هیچ خطایی کامپایل شود:
   ```bash
   npm run tauri build
   ```

#### روش دوم: حذف موقت آیکون اختصاصی ویندوز (پیش‌فرض سیستم)
اگر برای تست اولیه نیاز به بیلد سریع دارید و بودن آیکون اختصاصی روی فایل برایتان مهم نیست:
۱. فایل `src-tauri/tauri.conf.json` را با ردیتر متنی (مانند Notepad) باز کنید.
۲. خط زیر را در بخش انتهایی فایل پیدا کرده و حذف یا کامنت کنید:
   `"icons/icon.ico"`
۳. فایل را ذخیره کرده و دستور بیلد را دوباره اجرا کنید تا بلافاصله کامپایل موفقیت‌آمیز انجام شود!

## 🇺🇸 Brief English Summary

### Server Setup (Linux):
1. Install Node.js: `sudo apt install nodejs npm`
2. Prepare directory: `mkdir rustnet-server && cd rustnet-server` (Copy `standalone-server.js` here)
3. Install ws: `npm install ws`
4. Run in background: `PORT=3000 npx pm2 start standalone-server.js`

### Compiling Client on Windows:
1. Install Rust via [rustup.rs](https://rustup.rs/) (Choose default option 1 which uses Visual Studio MSVC C++ build tools).
2. Download this folder (using AI Studio menu's **Export to ZIP** or download file workflow), then open terminal on the extracted `rust-client/` directory.
3. Build optimal optimized binaries using:
   ```bash
   cargo build --release
   ```
4. Get your standalone native binary artifact at: `rust-client/target/release/rustnet-client.exe`. Execute to run!
